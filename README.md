# Acorn-globus
Created a Sales Analytic dashboard UI/UX of Dribbble
This project is a pixel-accurate analytics dashboard UI built as a frontend-only application, focusing on layout precision, component structure, and modern UI practices.
The dashboard visually represents sales performance, platform metrics, user activity, and trends using a clean, data-driven design.
